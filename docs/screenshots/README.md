# Screenshots

Drop README screenshots here as PNGs, then uncomment the grid in the root
`README.md` (and remove the "coming soon" line).

Suggested shots and filenames:

| Filename | What to capture |
|---|---|
| `suggestion-bar.png` | The pixel-art command-suggestion bar showing a proposed command. |
| `command-palette.png` | The command palette (`Ctrl+Shift+P`) open with a fuzzy query. |
| `themes.png` | A couple of themes side by side, or the theme switcher. |
| `quake.png` | Quake mode dropping in from the top of the screen. |
| `settings.png` | The native settings window. |

Guidelines:

- Use a clean, readable theme (e.g. Tokyo Night or Dracula).
- Capture at a normal window size; avoid huge 4K shots that dwarf the page.
- Prefer PNG; keep each file reasonably small (ideally < 500 KB).
- On Windows, the repo has a screenshot helper for the live window if you'd
  rather automate the capture.
