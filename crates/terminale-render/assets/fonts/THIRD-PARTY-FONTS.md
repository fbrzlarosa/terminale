# Third-Party Bundled Fonts

The following open-source monospace typefaces are embedded in the terminale
binary via `include_bytes!` and are shipped under their respective licenses.

| Family | SPDX License | Upstream |
|---|---|---|
| Ubuntu Mono | UFL-1.0 | https://design.ubuntu.com/font |
| Source Code Pro | OFL-1.1 | https://github.com/adobe-fonts/source-code-pro |
| IBM Plex Mono | OFL-1.1 | https://github.com/IBM/plex |
| JetBrains Mono | OFL-1.1 | https://github.com/JetBrains/JetBrainsMono |
| Inconsolata | OFL-1.1 | https://github.com/googlefonts/Inconsolata |
| Tabler Icons (subset) | MIT | https://tabler.io/icons |

The full text of each license is present in the corresponding subdirectory
alongside the font files:

- `ubuntu-mono/UFL.txt` — Ubuntu Font Licence 1.0
- `source-code-pro/LICENSE.md` — SIL Open Font License 1.1
- `ibm-plex-mono/OFL.txt` — SIL Open Font License 1.1
- `jetbrains-mono/OFL.txt` — SIL Open Font License 1.1
- `inconsolata/OFL.txt` — SIL Open Font License 1.1
- `icons/LICENSE` — MIT License (Tabler Icons)
